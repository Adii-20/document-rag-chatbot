# Document RAG Chatbot

A fully offline Retrieval-Augmented Generation (RAG) chatbot designed to answer natural-language queries against uploaded documents with high contextual accuracy across multi-page PDFs.

# Key Features
* Local & Cost-Free: Runs LLMs entirely locally via Ollama, eliminating external API costs and ensuring data privacy.
* End-to-End Pipeline: Fully designed data pipeline utilizing LangChain for document chunking, embedding generation, and vector search.
* Complex Document Handling: Extracts and cross-references data seamlessly across multi-page PDF inputs.

# Tech Stack
* Language: Python
* Frameworks: FastAPI, LangChain
* LLM Engine: Ollama
